# Логики создания ценности

Существует три фундаментальных логики создания ценности, к которым сводится любой бизнес-продукт: **Value Chain**, **Value Shop**, **Value Net** (Стабелл и Фьельдстад, 1998, на основе Томпсона, 1967).

Логики характеризуют **конкретные продукты**, а не отрасли. На одном рынке могут сосуществовать все три: конвейерная автомойка (Chain), детейлинг-студия (Shop), агрегатор моек самообслуживания (Net). Выбор логики — стратегическое решение компании, зависящее от её ресурсов.

По типу артефакта логику определяет блок 3 ДКЦП (см. [[dkcp-overview.md]]).

---

## Value Chain — Цепочка

**Суть:** трансформация ресурсов в тиражируемый продукт, который затем продаётся. Клиент не может или не хочет производить продукт сам.

**Схема (последовательно):**
1. Входящая логистика
2. Операции (производство)
3. Исходящая логистика
4. Продажи
5. Сервисное обслуживание

**Экономика:** добавленная стоимость = выручка − затраты на единицу. Работает **экономия на масштабе**: больше объём → ниже затраты на единицу.

**Примеры:** физические товары, цифровые инструменты без глубокой кастомизации (Miro, казуальные игры).

**Конкурентные барьеры:**

| Источник барьера | Пример |
|-----------------|--------|
| Уникальные поставщики / сырьё | ВкусВилл, McDonald's |
| Ноу-хау производства | Coca-Cola, KFC |
| Уникальная выходная логистика / продажи | Самокат, IKEA |
| Уникальное послепродажное сопровождение | Пожизненная гарантия Lynk&Co |

## Value Shop — Мастерская

**Суть:** диагностика и решение уникальной проблемы клиента при его непосредственном участии. Трансформируется не ресурс, а **сам клиент** — он приходит в одном состоянии, уходит в другом.

**Схема (циклически):**
1. Диагностика проблемы
2. Поиск способов решения
3. Выбор подходящего способа
4. Решение проблемы
5. Контроль результата

**Экономика:** экономии на масштабе нет. Работает **экономия на совмещении** (economy of scope) — один набор ресурсов решает разные проблемы.

**Важно:** продукт мастерской принципиально не тиражируем — создаётся под каждого клиента. Опыт участия клиента в процессе — часть воспринимаемой ценности.

**Примеры:** консалтинг, медицина, юридические услуги, дизайн-бюро, разработка ПО на заказ.

**Конкурентные барьеры:**

| Источник барьера | Пример |
|-----------------|--------|
| Уникальная технология диагностики | Институт Адизеса |
| Уникальные знания, опыт, репутация | McKinsey, Бюро Захи Хадид |
| Уникальная система контроля результата | Mayo Clinic |

## Value Net — Сеть

**Суть:** посредничество — создание и поддержка сети, участники которой платят за доступ к ней.

**Схема (одновременно и асинхронно):**
1. Расширение сети
2. Поддержка архитектуры сети
3. Оказание услуг сети

**Экономика:** ценность пропорциональна размеру сети. Работает **сетевой эффект**: больше участников → привлекательнее для новых. Сетевой эффект повышает готовность платить, а не снижает издержки (в отличие от масштаба).

**Критическая масса:** сетевой эффект начинает генерировать ценность только после её достижения — запуск сети затратен и долог.

**Ключевая проблема:** участники стремятся выйти на прямое взаимодействие, минуя посредника. Решение — оказывать услуги сети, которые ценны именно внутри неё (рейтинги, безопасность сделки, логистика). Ценность посредника выше там, где контакты одноразовые (такси, авито), и ниже там, где формируются постоянные отношения (клинер, психотерапевт).

**Примеры:** маркетплейсы, стриминг, банкинг, MMORPG, booking.com.

**Конкурентные барьеры:**

| Источник барьера | Пример |
|-----------------|--------|
| Уникальная инфраструктура | Booking, туристический кластер |
| Недостижимый размер сети | Ozon, Uber |
| Уникальные услуги сети | MMORPG-механики, преференции резидентам |

## Сравнительная таблица

| | Value Chain | Value Shop | Value Net |
|--|-------------|------------|-----------|
| Что трансформируется | Ресурсы → продукт | Клиент (его состояние) | Доступ к сети |
| Тиражируемость | Высокая | Нет (каждый раз уникально) | Высокая |
| Экономический эффект | Экономия на масштабе | Экономия на совмещении | Сетевой эффект |
| Ключевой ресурс | Средства производства | Знания, опыт, репутация | Связи, охват сети |
| Схема действий | Последовательная | Циклическая | Одновременная / асинхронная |
