# Верификация мотивационного конфликта

## Задача

Превратить формулировку МК в набор вопросов для интервью, чтобы подтвердить или улучшить её на основе данных от представителей КС.

---

## Алгоритм

**Шаг 1. Гипотезы опровержения**

Задай вопрос к МК: «Когда это может быть не так?» — то есть опиши реальность, в которой МК не существует.

Формат: «Если *[формулировка]*, то МК не существует, потому что *[объяснение]*»

Сформулируй не менее 3 гипотез-опровержений для каждой зоны неопределённости МК.

**Шаг 2. Исследовательские вопросы**

Для каждой гипотезы — один или несколько исследовательских вопросов: что нам как владельцам ДКЦП нужно прояснить, чтобы проверить гипотезу. Формулируй на уровне, понятном тебе (не для информанта).

**Шаг 3. Гайд для интервью**

На основе исследовательских вопросов сформулируй конкретные вопросы для информанта — оптимизированные для его языка и контекста.

**Шаг 4. Сборка документа**

Собери Markdown-документ:
1. Текущая формулировка МК
2. Сводная таблица: гипотезы опровержения + исследовательские вопросы
3. Гайд для глубинного интервью
